MegamanDx
=========

A Megaman tribute game using a 2d Game Engine that I will be using for other games, and I may make it a working framework.

Buttons
-------
W,UP   |     Move Up

S, DOWN  |   Move Down

A, LEFT   |  Move Left

D, RIGHT   | Move Right

Space    |   Jump

R         |  Punch?

F          | Fire!
